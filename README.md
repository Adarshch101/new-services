# React + Tailwind Website

Minimal Vite + React + Tailwind starter with Header, Hero, Services, Testimonials, Footer.

Quick start (PowerShell):

```powershell
cd "c:/Users/91870/Desktop/newProject"
npm install
npm run dev
```

Open the printed URL (usually http://localhost:5173).

Notes:
- Tailwind config and PostCSS are included.
- Edit `src/components/*` to customize content and styles.
